Getting Started

Here's how you can declare a variable in BCode:

var myNumber: 42

Control Flow

Using if-else statements in BCode:

if myNumber > 0 then
  log("Positive Number")
else
  log("Non-Positive Number")
end

Loops

Here's how you can use a simple for loop:

for i = 0 to 10 step 1 then
  log(i)
end
