Introduction

BubuCode is an easy, minimalistic, and interpreted programming language.

log("Hello, World!")

Variables

Declare variables using var:

var name: "Bubu"
var age: 20

Functions

Define functions using function:

function greet(name)
  log("Hello, " + name)
end

Conditionals

Use if statements for conditionals:

if age >= 18 then
  log("Adult")
else
  log("Minor")
end

Loops

BubuCode supports forloop and whileloop:

forloop i = 0, 5 do
  log(i)
end

Built-in Functions

Some useful built-in functions include:

  • log(): Outputs text to the console.
  • random(): Generates random numbers.
log("Random number: " + random(1, 10))
